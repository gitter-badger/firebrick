#Firebrick JS version: 0.13.8 [![Build Status](https://travis-ci.org/smasala/firebrick.svg?branch=master)](https://travis-ci.org/smasala/firebrick)

JavaScript MVC Framework built with:

* jQuery 2
* Knockout JS 3
* Require JS 2
